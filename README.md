# Currency Exchange 💱

A simple and responsive currency converter web app built using HTML, CSS, and JavaScript.
This project uses a free public currency exchange API to fetch real-time exchange rates.

---

## Features

- Convert currencies in real time
- Fetch live exchange rates using Fetch API
- Simple and clean UI
- Responsive layout using Flexbox
- Dynamic DOM updates
- Error handling for invalid requests

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

---

## API Used

This project uses the free public currency API:

https://github.com/fawazahmed0/exchange-api

Example endpoint:

```text
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/usd.json
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/akshitsingh51/currencyExchange.git
```

2. Open the project folder

```bash
cd currencyExchange
```

3. Run the project

Simply open `index.html` in your browser.
